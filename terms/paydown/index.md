## 2. Description of Service

PayDown is a mobile application that helps users create and follow a debt payoff plan. Users manually enter their debts (credit cards, loans, student debt), set a monthly available payment, and the App calculates a step-by-step payoff plan using the Snowball (lowest balance first) or Avalanche (highest interest first) method. Users can log payments and track their progress toward becoming debt-free. All data is stored locally on your device using Apple's SwiftData framework. No data is uploaded to external servers. The App does not connect to bank accounts or access external financial institutions.
