## 2. Description of Service

FocusBuddy is a mobile application that provides a Pomodoro-style focus timer with ambient soundscapes. Users can run timed work sessions (default 25 minutes) with short breaks (5 minutes) and long breaks (15 minutes), select ambient sound rooms (Café, Library, Co-working), and track their focus session history. All data is stored locally on your device using Apple's SwiftData framework. No data is uploaded to external servers.
